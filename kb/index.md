# Probe Ecosystem Knowledge Base

This KB is the source of truth for the probe tool ecosystem. The code is an implementation of the specifications described here. If the code contradicts this KB, the code is wrong.

## How to use this KB

1. Start here. Read the section you need below.
2. Follow cross-references to load specific files.
3. Each section has its own `index.md` if it contains multiple files.
